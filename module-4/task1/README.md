Configuration:

ESP32 Pins used for circuit - GPIO 19, 18, 5, 17, 12

Stepper Motor:

- Powered from 12V power supply
- Pins 1N1-1N4 connected to ESP32 GPIO pins 19, 18, 5, 17

Servo Motor:

- Powered by 3V3, GND from ESP32
- Control input connected to ESP32 GPIO 12
